## GitHub Pages 배포

- **사이트 URL**: https://kkdndd.github.io/formi-landing/

### GitHub Pages 활성화 방법

1. 저장소 **Settings** 탭 이동
2. 왼쪽 메뉴에서 **Pages** 선택
3. **Build and deployment** 섹션에서:
   - **Source**: Deploy from a branch
   - **Branch**: main
   - **Folder**: / (root)
4. **Save** 클릭

몇 분 후 위 URL에서 사이트를 확인할 수 있습니다.

## 로컬 실행

`index.html`을 브라우저에서 직접 열어 확인할 수 있습니다.
